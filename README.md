# Sales Data Analysis & Forecasting

A Retail Analytics project that analyzes historical sales data to identify trends, seasonality, and the impact of discounts on revenue.  
The project implements regression and time series techniques to forecast future sales.

---

## Project Overview

This project focuses on exploring and forecasting retail sales:

- Analyze monthly sales trends  
- Visualize seasonality patterns  
- Examine correlation between discounts and sales  
- Predict future sales using regression models  
- Evaluate model performance using MAE and RMSE  

---

## Dataset Information

**Features:**  

- Date  
- Product Category  
- Discount Percentage  
- Revenue  

The dataset is used to train regression models for accurate sales predictions.

---

## Key Tasks Performed

- Monthly sales trend analysis  
- Seasonality visualization  
- Correlation analysis between discounts and sales  
- Sales prediction using Linear Regression  
- Model evaluation using **MAE** (Mean Absolute Error) and **RMSE** (Root Mean Squared Error)
